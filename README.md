## 1.사용 Api
- 나이스 교육정보 개방포털에서 학교 급식 식단 정보 받아서 사용.
- openweathermap.org Api사용으로 날씨정보 출력.

## 2.구동
- 나이스 교육정보 포탈에서 받아온 xml파일을 파이썬 라이브러리 xml.etree와 bs4를 사용하여 텍스트로 변환시키고 출력시킨다.
